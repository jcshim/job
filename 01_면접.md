# ✅ 1. 기본 준비 (필수 체크리스트)

## ✔️ 1) 회사·직무 분석

* **왜 이 회사를 선택했는가?**
  → 기업 미션·비전·사업 구조 파악
* **이 직무를 왜 지원했는가?**
  → 직무 정의, 필요 역량, 최근 기술 트렌드 정리
* **경쟁사 대비 차별점**
  → 같은 업종 3개 정도 비교 후 핵심 특징 정리

**준비물:**

* 회사 최신 뉴스 3개
* 직무 기술서(Job Description, JD) 요약본
* 관련 경험과 역량 mapping sheet

---

# ✅ 2. 자기소개서 & 경험 정리

## ✔️ 1) 자기소개(1분 기반)

* 핵심 키워드 2~3개만 제시
* 경험 나열 금지 → “나는 어떤 사람인가” 중심
* 결론 → 근거 → 사례  구조로 정리

## ✔️ 2) STAR 기반 경험 정리

모든 경험(School project, 인턴, 팀플 등)을 아래 4단계로 정리:

* **S**ituation: 어떤 상황에서
* **T**ask: 역할/목표
* **A**ction: 구체적인 행동
* **R**esult: 결과와 배운 점

# ✅ 3. 예상 질문 준비

## ✔️ 1) 기본 질문

* 자기소개 해주세요
* 지원동기는 무엇인가?
* 우리 회사에서 이루고 싶은 목표는?
* 본인의 강점/약점은?
* 갈등 상황 해결 경험?
* 입사 후 5년 후 모습?

## ✔️ 2) 직무 기반 질문

* 전공지식 (예: 컴퓨터공학 → 자료구조, OS, 네트워크)
* 실제 문제 상황 가정 질문
* 수행했던 프로젝트의 역할, 성능/결과, 어려움 해결 과정

## ✔️ 3) 압박/논리 질문

* “그건 다른 지원자도 할 수 있는 거 아닌가요?”
* “만약 성과가 안 나오면 어떻게 하겠습니까?”
* “왜 성적이 별로인가요?”

→ **논리적으로 설명 + 대안 제시**가 핵심

# ✅ 4. 포트폴리오 / 프로젝트 정리 (기술직무 필수)

## ✔️ 1) 발표용 슬라이드

* 5~7분 분량으로 압축
* 문제 → 접근방법 → 기술스택 → 역할 → 성과
* 성능 비교 그래프나 시각화 자료 필수

## ✔️ 2) 깃허브/노션 정리

* 프로젝트 Readme 구성:

  * 개요
  * 기술 스택
  * 문제 해결 로직
  * 결과
  * 코드 링크
  * 시연 영상

---

# ✅ 5. 면접 매너 & 태도

## ✔️ 1) 외적 준비

* 단정한 복장
* 지나친 향수 금지
* 이력서/포폴 출력본 + 펜 지참
* 10~15분 일찍 도착

## ✔️ 2) 말하기 태도

* 예/아니오보다 **근거 중심 설명**
* 질문 이해 안 될 때: “다시 한 번만 말씀 부탁드립니다.”
* 질문에 바로 답 못하면 **생각할 시간 요청 가능**

  * “잠시 생각해보겠습니다.”

# ✅ 6. 마지막 질문(역질문) 준비

면접관에게 반드시 질문할 한 가지:

* “저의 이력 중에서 보완하면 좋을 점이 있을까요?”
* “이 직무에서 뛰어나기 위해 필요한 핵심 역량은 무엇인가요?”
* “팀은 어떤 방식으로 협업하나요?”

**절대 하면 안 되는 질문**

* 연봉, 야근, 복지, 휴가 우선 질문
* 홈페이지에 이미 있는 내용 질문

# ✅ 7. 멘탈 & 컨디션 관리

* 면접 하루 전: 자기소개/포트폴리오만 가볍게 복습
* 충분한 수면
* 면접 직전에는 발표 흐름만 체크
* 면접 후 복기: 질문 목록 기록하고 개선 가능성 적기
